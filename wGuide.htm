Guide

(Based on Android)

Download and install WhatsApp Sign Up Interface Add or Block Friend Call A Person

Download and install WhatsApp


Go to the Play Store and find "whatsapp" on search bar

Click the "Install" button to download WhatsApp

Back To Top

Sign Up

(images from WonderHowTo)


Open the app and click the "Agree and Continue" button

Input your phone number and click OK

Wait for the SMS to verify your account

Click "Continue" and you can start the use WhatsApp

Back To Top

Interface


This is chat area. Click icon to enter chat box

This is call history. You can click the call button to call the selected person

This is contact list. You can see your friend's profile after clicking them

This is text box. You can either communicate with your friend with text messages, images, voice message and phone call.

Back To Top

Add or Block Friend


To add a person, go to the contact list and click add button

Input information of the person you want to add to your phone contact

To block a person, go to his/her chat box and click menu

Select "More"

Select "Block"

Select "Ok" and you will block the person

Back To Top

Call A Person


Select the person you want to call in contact list

Select call button

You will then call the person
